Mediatyper

Publisert 01. oktober 2018.

CSS mediatyper og queries, er en måte å bestemme hvilket stilsett man skal bruke på nettsiden basert på forskjellige faktorer. Man kan ha egne stilsett for visning på skjerm, egne for ting som skal skrives ut og eget stilsett for tekst til tale. Det er også mulig å bestemme stilsett basert på for eksempel: Bredden på skjermen som viser nettsiden, om siden vises i portrett- eller landskapsmodus eller om man overstiger en minimumsbredde.

Per i dag benyttes følgende mediatyper:

  • all - for alle mediatyper
  • print - for ting som skrives ut
  • screen - for ting som vises på skjerm
  • speech - for tekst til tale

Det finnes flere eldre mediatyper, men disse brukes i liten grad og vil etter hvert ikke støttes, så de omtales ikke her.

Ved å bruke mediatyper og queries kan man skreddersy brukeropplevelsen basert basert for hvordan de bruker tjenesten og ut ifra hvilken enhet de bruker. Ulempen med dette kan være at man for kan ende opp med en mye større kildekode dersom man skriver egen kode for hvert tilfelle, i motsettning til å heller lage for ekspel responsivt design.
